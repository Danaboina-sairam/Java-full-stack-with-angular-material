1.we have array of 0 and 1 then keee 0  at last and 1 at first

Normal Java:

public static void moveZeroes(int[] arr) {
    int count = 0;

    for (int num : arr) {
        if (num == 1) count++;
    }

    for (int i = 0; i < arr.length; i++) {
        if (i < count)
            arr[i] = 1;
        else
            arr[i] = 0;
    }
}

Java Streams:

int[] arr = {0,1,0,1,1,0,1};

int[] result = IntStream.concat(
        Arrays.stream(arr).filter(i -> i == 1),
        Arrays.stream(arr).filter(i -> i == 0)
).toArray();
